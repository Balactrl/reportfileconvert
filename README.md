Sales Converter

This folder contains the Streamlit app and supporting module.

Files:
- `liteposextotxto.py` - main Streamlit app
- `ELTester.py` - conversion helper functions
- `requirements.txt` - Python dependencies

Deployment:
- Set Streamlit "Main file" to `convertor/liteposextotxto.py` in Streamlit Cloud.
- Ensure `requirements.txt` is present at repo root or in same folder.
